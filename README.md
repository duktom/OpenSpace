# OpenSpace
For the application to function properly, you must:

1. Create a PostgreSQL database (without tables)
2. Create an .env file based on the .env.example file and complete it
3. Create venv by "python3 -m venv venv" then "source venv/bin/activate"
4. Install requirements from requirements.txt by "pip install -r requirements.txt"

To run an application you need to type "python3.12 main.py"
